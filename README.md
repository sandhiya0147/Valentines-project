# Valentine Proposal Website

A fun and interactive Valentine’s Day proposal website built to express love in a playful and creative way. This project uses simple web technologies to create a delightful user experience with animations, styling, and a happy ending.

---

## Features

- Cute and romantic UI design

- Custom CSS styling

- Fun images and animations

- Interactive button navigation

- Fast and lightweight static site

- Deployed on Render

---

## Tech Stack

- HTML – Page structure

- CSS – Styling and layout

- JavaScript – Interactivity

- Hosting – Render (Static Site)

- Version Control – Git & GitHub
